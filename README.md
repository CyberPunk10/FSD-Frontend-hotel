# Webpack-02-Pug (в разработке)
## перед запуском нужно выполнить 
1) `npm install`
2) `npm run dev`
готовые файлы появятся в папке `dist`

Из .gitignore убрана папка 'dist/', которая, теперь, является корнеыой для GitHub Pages
